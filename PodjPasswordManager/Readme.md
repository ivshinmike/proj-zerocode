# Password Manager (CLI, Python)

CLI-приложение для безопасного хранения паролей с использованием:
- SQLite3
- SHA-256 для мастер-пароля
- Fernet (cryptography) для шифрования паролей

## Возможности
- Мастер-пароль (запрашивается при первом запуске)
- Все пароли хранятся **только в зашифрованном виде**
- Генерация надёжных паролей
- Добавление, просмотр, удаление записей
- Отображение списка сохранённых паролей (без раскрытия пароля)
